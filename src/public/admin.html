📊 Portfolio Admin

Manually add your deployed tokens, NFTs, and collections to the portfolio cache

🪙 Add Token to Cache

This adds a token deployment to your portfolio. Fill in the details from your Token Factory contract deployment.

🖼️ Add NFT to Cache

This adds an NFT mint to your portfolio. Use the token ID, metadata URI, and recipient address from your mint transaction.

🏛️ Add Collection to Cache

This adds a collection to your portfolio. Use the name, symbol, and base URI from your Collection Factory deployment.

📝 Current Cache

ℹ️ About This Tool

The portfolio cache stores information about tokens, NFTs, and collections you deploy through the chat modals. If you deployed contracts before this cache was created, use this tool to manually add them so they appear in your portfolio.

After adding your deployments here, go to 📊 Portfolio and refresh the page to see them appear.
