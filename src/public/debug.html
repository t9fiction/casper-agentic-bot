Troubleshooting Steps

  1. Check Wallet Connection:
    • Click "Check Wallet" to verify extension is installed
    • Click "Connect Wallet" to establish connection
    • Check console (F12) for any errors
  2. Verify Backend Config:
    • Click "Check Backend Config" to verify server configuration
    • Ensure WALLET_ACCOUNT_HASH is set in .env
  3. Check Portfolio Data:
    • Click "Fetch Portfolio Data" to see what the API returns
    • Look at "deployed_contracts" array - should show contracts from blockchain
    • Look at "custom_tokens" - should show tokens logged via chat
  4. If Still Empty:
    • Check if contracts were actually deployed to testnet
    • Verify CSPR_CLOUD_API_KEY is valid
    • Go to Admin Dashboard to manually add deployments
