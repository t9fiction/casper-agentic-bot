## KEY FINDINGS

### AI System Architecture
- **Framework:** LangGraph + LangChain with GPT-4o-mini
- **Tools:** 7 available tools including contract calling
- **Memory:** Session-based conversation history (last 20 messages)
- **Validation:** Pre-execution parameter validation
- **Contracts:** 3 smart contracts (Token Factory, NFT Marketplace, Collection Factory)

### Smart Contract Integration
- **NFT Marketplace:** Handles individual NFTs with custom metadata URIs
- **Collection Factory:** Handles NFT series with auto-generated URIs
- **Token Factory:** Handles custom token deployment and management

### Conversation Memory Details
- Tracks collection IDs across messages
- Remembers contract names and addresses
- Enables multi-step workflows in single conversation
- Stores up to 20 previous messages for context

### Entry Points Available
**NFT Marketplace (11 entry points):**
- mint_nft, transfer_nft, list_nft, unlist_nft, buy_nft
- nft_info, owner_of, total_nfts, metadata_uri

**Collection Factory (9 entry points):**
- create_collection, mint_nft, transfer_nft, list_nft
- collection_info, total_collections, total_nfts_in_collection
- nfts_by_collection, metadata_uri

**Token Factory (6 entry points):**
- deploy_token, transfer, balance_of, token_info, total_tokens, mint
